# htc-bim-library
HTC BIM - Parametric Component Library

This repository contains parametric components for modeling patio covers, pool cages, sunrooms, glass enclosures, window and door replacement, and decks
  1. Insulated roof and sidewall panels (aluminum clad EPS Core self-supporting panels)
  2. Aluminum extrusion framing (columns, beams, framing members)
  3. Lumber and connectors (decks, railings, anddoor/window framing)
  4. Facades (house walls, eaves, soffit, fascia)
  5. Foundation (slab, footers, deck posts)

These components are free to download and use for any purpose without additional permission.

Cheers!
